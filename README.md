1. Что такое репозиторий?
  Репозиторий - Хранилище данных, в нем может храниться различные данные, от папок до файлов, к примеру, кода
2. Особенности репозитория
  Удобен для хранения данных, или для совместной работы с файлами
3. Что такое команда разработки?
  Команда разработки - Группа людей, создающих один, заранее определенный проект. В ней есть: глава разработки (тимлид), тестировщик, разработчик, аналитик и др. роли
4. Связи в команде разработки
  Все роли непосредственно связаны между друг другом. Первым делом аналитик выслушивает заказчика, и передает "эстафетную палочку" разработчику, он разрабатывает сам проект, его код и сущность. Практически одновременно с разработчиком работает тестировщик,
  он тестирует программу на работоспособность, создает тест-кейсы и передает их разработчику, чтобы он в свою очередь ввел необходимые поправки. И конечно же ими всеми заправляет тимлид.
